# uBid

A custom ecommerce design built on top of Bootstrap 4

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

Download from source:
```sh
git clone https://github.com/DesignGoodDev/ubid.git
```

Install:
```sh
cd ubid
npm install
```

## Usage

Develop locally:
```sh
npm run dev
```

Build for production:
```sh
npm run build
```
